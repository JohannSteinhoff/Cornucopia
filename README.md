# The Cornucopia — A Natural Foods Market

Website for **The Cornucopia**, San Marcos' beloved natural foods market — organic produce, supplements, bulk goods, body care and more, locally rooted for over thirty years.

## 🌿 Live site

**https://johannsteinhoff.github.io/Cornucopia/**

## About

A static marketing site (plain HTML/CSS) hosted on GitHub Pages.

- `index.html` — page markup
- `styles.css` — styles
- `assets/` — site assets (logo, hero, banners)
- `images/` — product, storefront, and interior photography

## Local development

Open `index.html` directly in a browser, or serve the folder:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```
